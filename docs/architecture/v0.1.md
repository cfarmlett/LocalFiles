Implemented and hardened V1 workflows:

- Merge PDF
- Split PDF
- Reorder Pages
- Rotate Pages
- Delete Pages
- Metadata Removal

V1 polish still planned:

- ZIP export for Split PDF

Explicitly excluded from V1:

Browser redaction is omitted intentionally because the current browser-only
stack does not meet the project's definition of successful redaction. Future
desktop or native research may revisit redaction with a stronger verification
model.
