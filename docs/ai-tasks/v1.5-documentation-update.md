Current V1.5 scope decisions:

### Validation Philosophy

Invalid input should be blocked with a specific validation message.

Examples:

- Split range references page 30 in a 20-page PDF.
- Delete page outside document bounds.
- Similar objectively invalid user input.

Valid but low-value operations should generally remain allowed.

Examples:

- Merge one PDF.
- Split a one-page PDF.
- Reorder a one-page PDF.
- Run Metadata Removal when no removable standard metadata is detected.
